Бот для управленческого учета остатков на складе.

Проект реализован через автоматизированную таблицу в Google Sheets, к которой бот подключается и с которой работает.

Сам бот реализуется через 3 файла: main.py, sheet_handler.py, markups.py. Первый содержит главную логику телеграм-бота,
второй -- логику управления гугл-таблицей, третий является сборником всех возможных кнопок (данный файл создает все
нужные кнопки сразу после запуска бота)

Логика бота реализуется через библиотеку telebot и технику состояний, а логика гугл-таблицы реализуется через библиотеку
pygsheets.

Сама таблица сильно вдохновлена из приложенного в задании примера (https://www.youtube.com/watch?v=5EkiQjAB_wg) и большая часть логики в программе
выполняется именно ей. Она содержит необходимые справочники в качестве листов. В таблице есть скрытые листы, облегчающие логику программы (например 'Все сырье', объединяющее справочник готовых изделий и справочник материалов и полуфабрикатов в один)

Бот и таблица не предназначены для использования несколькими пользователями одновременно.
